If you sell through Magento, you can start selling on Wanderset in seconds.
We make connecting your Magento store to Wanderset a snap.

how it works

  • We automatically load your Magento product inventory into Wanderset and keep it updated in realtime.
  • When customers order your products on Wanderset, orders appear in your Magento account.
  • Ship your orders directly to the customer and collect payment from Wanderset.

how to connect your store with wanderset

  • To get started, go to your Magento store, log in as an admin and hover over "System" tab in the navbar.
  • Then hover over "Web Services" and click on "SOAP/XML-RPC - Roles".
  • When the page opens, click on "Add New Role".
  • Enter role name. For example, "Wanderset" and click on "Role Resources"
  • Choose "All" from "Resource Access" dropdown and click on "Save Role" button.
  • After that hover again over "System" tab in the navbar and over "Web Services" and click on "SOAP/XML-RPC - Users" button.
  • Click on "Add New User" button.
  • In the new user form, fill in the following data and click on "User Role" button in the sidebar:
    User Name: Desired name for the user (e.g. Wanderset)
    First Name: Desired first name for the user (e.g. Wanderset)
    Last Name: Desired last name for the user (e.g. Store)
    Email: support@wanderset.com
    API Key: Any combination of letters and numbers
    API Key Confirmation: repeat API Key
    This account is: ACTIVE
  • Select the user role created in previous steps and click on "Save User" button.
  • Copy user name and API key entered in the form in the previous step and paste it in the form below. Click on Submit and the integration is complete.
